# Prueba de Unidad

En esta prueba de unidad de JUNIT 4 se aplican conceptos de QA-Testing a partir de un SRS (documento de requerimientos), la documentacion de las clases a testear, y el archivo .jar compilado con las clases.

### Conceptos aplicados

Test de Caja negra: Se realiza el test de caja negra para cada modulo del sistema que sea necesario.
Test de Persistencia: A nivel conceptual, se levanta el sistema entero para leer y se escribe todo al mismo tiempo (no hay base de datos)
Test de Excepciones: Se testea el correcto funcionamiento de las Excepciones del sistema
Test de Integracion: Se respeta la integracion de manera limpia y clara
